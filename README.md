# scaladoc-demo

## 快速开始

```bash
sbt doc
```

生成的文档参见 target/scala-2.12/api 文件夹。

## 温馨提示
- scala 3 开始支持 markdown 格式的文档注释，详情参见
  [New features for scaladoc](https://docs.scala-lang.org/scala3/scaladoc.html)。
